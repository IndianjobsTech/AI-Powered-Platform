# Freebuff Web Dashboard

Next.js 16 web application for managing business automation.

## Pages

- `/` - Auto-redirects to dashboard or login
- `/login` - Sign in
- `/signup` - Create account
- `/forgot-password` - Password reset
- `/dashboard` - Business overview with stats
- `/whatsapp` - WhatsApp messaging interface
- `/leads` - Lead management
- `/appointments` - Appointment management
- `/analytics` - Business analytics
- `/billing` - Subscription plans
- `/settings` - AI settings & configuration

## Tech

- Next.js 16 (App Router)
- Tailwind CSS 4
- Firebase Auth
- Recharts for analytics
- shadcn/ui component system
